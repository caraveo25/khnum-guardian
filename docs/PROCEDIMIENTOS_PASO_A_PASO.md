# Procedimientos Paso a Paso para Importaciones

## Importaciones desde Estados Unidos (USMCA)
**Tiempo Estimado: 20-21 Días**

### 1. Preparación Pre-embarque
   - **Selección de Proveedores**
   - **Negociación de Términos**
   - **Documentación Necesaria**
     - Factura Comercial
     - Bill of Lading
     - Certificado de Origen

### 2. Embarque
   - **Organización del Transporte**
   - **Seguro de Carga**

### 3. Envío
   - **Seguimiento del Envío**
   - **Documentación de Envío**

### 4. Despacho de Aduanas
   - **Documentación para Aduanas**
   - **Pagos de Aranceles y Impuestos**

### 5. Recepción y Entrega
   - **Verificación de la Carga**
   - **Almacenamiento y Distribución**
