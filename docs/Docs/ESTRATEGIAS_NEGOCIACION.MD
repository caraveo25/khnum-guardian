# Estrategias de Negociación

## 5 Técnicas Probadas

### 1. Anclaje (Anchoring)
- Establece precio inicial bajo
- **Ahorro:** 12%
- **Tiempo:** 5-7 días

### 2. Flinch (Sorpresa)
- Reacciona con sorpresa al precio
- **Ahorro:** 15%
- **Tiempo:** 3-4 días

### 3. Salto de Rana (Leapfrog)
- Incrementos progresivos pequeños
- **Ahorro:** 10%
- **Tiempo:** 7-10 días

### 4. Concesiones Limitadas
- Ofrece valor por precio mejor
- **Ahorro:** 8%
- **Tiempo:** 10-14 días

### 5. Ultimátum
- Oferta final decisiva
- **Ahorro:** 18%
- **Tiempo:** 1 día
